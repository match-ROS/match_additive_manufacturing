# Asset instructions

## Main video
The landing page uses a YouTube embed.
Edit the iframe URL in `index.html` if you want to switch to another video.

## Landing-page image stack
Put your 11 JPG files here:

- `assets/landing/landing-01.jpg`
- `assets/landing/landing-02.jpg`
- ...
- `assets/landing/landing-11.jpg`

Then replace `.svg` with `.jpg` inside the landing-page stack in `index.html`.

## Planning assets already inserted
These files are already included:

- `assets/planning/initialPose.png`
- `assets/planning/initialPoseCorner.png`
- `assets/planning/scene.gif`

## Chapter pages
Use these folders:

- `assets/printing-process/`
- `assets/trajectory-planning/`
- `assets/trajectory-retiming/`
- `assets/robot-control/`
- `assets/evaluation/`

For each folder, replace the placeholder SVGs with JPGs of the same base name and update the file extensions in the corresponding html page.

## Recommended image sizes
- landscape figures: about 1600 px wide
- jpg quality: around 80–85%
- gifs: compress if they are larger than needed
